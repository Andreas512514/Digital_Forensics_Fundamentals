# Windows Forensics

## FTK Imager

FTK Imager is a widely used tool for taking disk images of Windows operating systems.
It offers a user-friendly graphical interface for creating the image in various formats. This tool can also analyze the contents of a disk image.
It can be used for both acquisition and analysis purposes.<br><br><br>


## Autopsy

[Autopsy](https://www.autopsy.com) is a popular open-source digital forensics platform. An investigator can import an acquired disk image into this tool, and the tool will conduct an extensive analysis of the image.
It offers various features during image analysis, including keyword search, deleted file recovery, file metadata, extension mismatch detection, and many more.

## DumpIt

[DumpIt](https://www.toolwar.com/2014/01/dumpit-memory-dump-tools.html) offers the utility of taking a memory image from a Windows operating system. This tool creates memory images using a command-line interface and a few commands. The memory image can also be taken in different formats.
